# 2026-07-02 | Docker Compose Sections
- - -
В конфигурационном файле **Docker Compose** есть 3 основные секции, которые используются в большинстве случаев:

- **services** - настройка контейнеров
- **volumes** - настройка хранилищ
- **networks** - настройка сетей, чтобы контейнеры могли взаимодействовать в рамках одной сети и обращаться друг к другу по **DNS**

Но есть ещё секции:

- **secrets** - для безопасной передачи секретных данных(ключи, сертификаты) в контейнер в виде файлов, а не переменных окружения(которые могут перехватываться)
- **configs** - похожа на **secrets**, но предназначена для открытых конфигурационных файлов, чтобы не монтировать их через **volume** 
- **include** - позволяет разделять огромные compose файлы на несколько более маленьких и импортировать их друг в друга.
